English

I am studying Python programming because I want:

* To learn a trade that allows me to work remotely;
* To generate an income that allows me to have enough money to:

1. Be able to buy a farm with at least 1 hectare;
2. Have money to carry out necessary renovations;
3. Make my farm self-sufficient and automated;
4. Secure a future;
6. Bring more happiness to the person I love.

Notes:
1. ### were written with the assistance of AI.
2. ## means it is an exercise statement.
3. # are simple comments.
4. Names starting with "f_" are my functions.
5. Names starting with "pp_" are my small projects.
6. Names ending with "#" are unsolved exercises.

Português 

Estou estudando programação em Python porque quero:

* Poder aprender um ofício que me permita trabalhar a distância;
* Gerar uma renda que me permita ter dinheiro o suficiente para:

1. Poder comprar uma quinta com pelo menos 1 hectare;
2. Ter dinheiro para realizar reformas necessárias;
3. Tornar a minha quinta se torne autossuficiente e automatizada;
4. Conquistar um futuro;
5.Trazer mais felicidade à pessoa que eu amo.

Notas:
1. ### foram escritos com auxílio de IAs.
2. ## significa que é um enunciado de exercício.
3. # são comentários simples.
4. nomes que começam com "f_" são as minhas funções.
5. nomes que comecem com "pp_" são os meus projetos pequenos.
6. nomes que terminem com "#" são exercícios não resolvidos.
